# OpenclawAgents

OpenClaw **workspace** (skills + agent context) lives in **`workspace/`**.

- **Requirements:** [`SkillsToAdd.md`](./SkillsToAdd.md)
- **Deploy to server:** [`workspace/SKILLS_SYNC.md`](./workspace/SKILLS_SYNC.md) → sync to `~/.openclaw/workspace/` on the gateway host

Existing QuizFactor pipeline skills: `workspace/skills/qf-*`. New sub-agents from SkillsToAdd: `workspace/skills/social-*`, `product-manager-agent`, `auto-research-agent`, `auto-video-generation`, `auto-image-generation`, `reddit-trend-poster`, `adverts-creator`, `tiktok-video-ads-creator`, `linkedin-article-writer`, `multi-agent-orchestrator`.
